Quantization Settings

Matrix weights use int8; layer norms stay FP32. The file also holds the tokenizer block and headers, so total size is not a fixed multiple of “FP32 model size.” To shrink further: smaller vocabulary / smaller model, or a larger LittleFS partition (KB above). This format (v2 for GPT-2 family includes LayerNorm bias tensors) only supports FP32 or INT8 weights on device—INT16 would need new firmware.
